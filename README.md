# corporate_analysis
기업의 재무제표를 분석

# 실행환경
- python 3.10
- pyqt5

# 화면

# 설명
1. https://opendart.fss.or.kr/
    - DART에서 재무정보 일괄다운로드(csv 파일)
    - 올바른 경로에 해당 파일 위치시키기
2. ebest, kiwoom 가입 후 api 사용 등록
3. 기업 분석 진행

# 향후 개발 방향
1. ui 업데이트
    - 재무제표 업로드 연결
    - 재무제표와 연결하여 기업 분석
    - 투자 종목 선정
2. 투자 리포트 기능 추가
